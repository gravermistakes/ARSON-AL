# Web3 Audit Report Archives

## Audit Finding Databases and Indexes

- **Solodit** (`solodit.xyz`): Curated search engine containing thousands of individual, tagged smart-contract audit findings. Categorized by impact, bug class, and language. Excellent for checking if an identified finding pattern has real-world high/medium severity precedents.
- **Audit Hero** (`audithero.com`): Meta-search database for public audit reports and findings.
- **`0xNazgul/Blockchain-Security-Audit-List`**: Comprehensive Git repository indexing public audit reports and security assets.

---

## Firm and Platform Audit Portfolios

- **Consensys Diligence Audits**: Highly detailed, structural security audits of premier DeFi applications.
- **Code4rena Reports**: Public archive of competitive audit results. Shows distinct duplicates, warden disputes, and judging outcomes. Excellent for checking gate G5 (False-Positive) filters.
- **Sherlock Reports**: Public portfolios of audited targets, detailing consensus outcomes and Watson submissions.
- **Spearbit Portfolio PDFs**: Technical smart contract audit reports on complex systems (e.g. Layer 2, bridges, custom protocols).
- **QuillAudits Reports**: Archive of decentralized application and token audit findings.
