# Guild Hall Campaign Mode Orchestration

The quest/objective/points/badges/tiers/leaderboard gamification model:
- Engagements = Quests (objectives to pursue)
- Points = Severity scores (P1=100, P2=50, etc.)
- Chains = Multi-step attack paths with efficiency bonuses
- Leaderboard = Actor/seed coverage tracking and scoring

Campaign-Mode NPC orchestration manages:
- Quest lifecycle (definition, execution, completion)
- Reward calculation (points, badges, tier progression)
- Leaderboard state management
- Team coordination (multi-player quest)

See .campaign/ model in guild-hall for implementation templates.
